北交打新记账本 使用说明

一、功能概述
本应用是一款面向北交所打新投资的记账工具，完全基于浏览器本地存储，无后端、无需安装。

二、主要功能
1. 投资记录：记录每次打新的投入金额、中签股数
2. 转账记录：记录参与者之间的资金往来
3. 资金记录：查看各人的账户余额变化
4. 收益结算：自动计算每人收益和结算金额

三、数据安全
- 所有数据存储在浏览器本地 IndexedDB 中
- 建议定期通过"转账记录"页面的导出功能备份数据
- 换设备或清缓存会导致数据丢失，请提前备份

四、版本更新
每次更新会在 GitHub 上发布，导入新版本时建议先导出备份。
